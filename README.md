# blogdecafe
This project was made to practice HTML and CSS with Responsive Web Design using Flexbox and Grid. It is a blog where there is a home with posts and three more sections. About us, courses and Contact.

## Tools used:
- Normalize CSS
- Modernizr
- Google Fonts
- ResponsivelyApp
